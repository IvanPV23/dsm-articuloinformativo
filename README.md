## 🛠️ Tecnologías utilizadas

- **Lenguaje**: Kotlin
- **Interfaz**: Jetpack Compose
- **Herramientas**: Android Studio
- **Gestión de dependencias**: Gradle

## 📋 Requisitos del sistema

- Android Studio Arctic Fox o superior
- SDK Android 7.0 (Nougat) o superior
- Gradle 7.0+
